Para instalar las dependencias empezamos creando un entorno virutal por seguridad con el 
siguiente codigo en la terminal : python -m venv .venv
despues de eso ejecutamos en terminal pip install -r requirements.txt para que instale
las dependencias que use para el ejercicio, en este caso jupyter lab.
Para ejecutar el program hay que usar 'jupyter lab' en la terminar para que abra en el navegador
jupyter y poder correr el programa.
